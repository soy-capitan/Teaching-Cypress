Tags

Tags are the basic structure of HTML documents. They are the hidden keywords within a web page that define how your browser must format and display content. Most tags must have two parts, an opening and a closing part.
The opening and closing brackets for html are < > . For instance, to begin a tag use "< name of tag >" and to end a tag use "< / name of tag >". You don't want any spaces within the tags.
They tell the browser how to view what kind of thing you are programing.
ex. img = image Tells the browser to render an image.

The br tag inserts a single line break

Attributes

Attributes are extra bits of info that define characteristics or properties of the element such as width/height or font/color of an image. Attributes are always specified in the start tag (or opening tag) and usually consist of name/value pairs like name="value" . Attribute values should always be enclosed in quotation marks. Anything after the = is an attribute of the tag
ex. "font face="arial" "color=red" div - divide, id - identify

    Lists

    • "ul" - Unordered list: used to group a set of related items in no particular order using bullet points/characters
    • "ol" - Ordered list: used to group a set of related items in a specific order using numbers

    • "dl" - Description list: used to display name/value pairs such as terms and definitions.
      • Use "dl" to start the list. "dt" (define term) for the element or term and then "dd" for the description
